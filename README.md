# Fake News Detector

## Description
This project checks whether a given news text is fake or real using simple keyword-based logic.

## Features
- Detects fake news using common keywords  
- Checks for emotional words  
- Shows reasons for the result  
- Simple and easy to use  

## Requirements
- Python 3 installed  

## How to Run

1. Open PowerShell  
2. Go to the project folder:
   cd Desktop  
   cd fake-news-detector  

3. Run the program:
   python main.py  

## Usage

- Enter any news text  
- The program will show:
  - Result (Fake or Real)
  - Reasons for classification  

## Example

Input:
Breaking shocking news!

Output:
Likely Fake News  
- Keyword detected: breaking  
- Keyword detected: shocking
